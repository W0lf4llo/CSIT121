My UI/UX Evaluation of Nasa's Website!

Overall Evaluation

Screenshot of The Website

Website Information

Website URL: https://www.nasa.gov

Name of Website: NASA - National Aeronautics and Space Administration

Target Audience: Students, educators, space enthusiasts, and the public.

How is The Website Organized

The website has a navigation bar up at the top and it's very easy to use in my opinion. The options are all neatly laid out and if you can't find a specific option there's a search bar which will help you out drastically!

CRAP Design Principles

Contrast: The website uses plain white text on dark backgrounds to make it easy to read.

Alignment: The content is very neatly aligned in its columns.

Proximity: All the content relating to eachother is grouped together which helps users find what they're looking for!

Accessibility Checker Results

  • One link is missing text
  • Three buttons don't have labels at all
  • One button is way to small to click
  • One issue with the color contrast was discovered

Overall Usability of The Website

Effectiveness: The website is very effective as it allows the users to read any upcoming news, watch uploaded videos, and explore many missions they've done over the past few years.

Efficiency: The website is very efficient as it allows users to easily search throughout the website to find something in particular with the built in search bar and navigation tab.

Engagement: The design of the website allows for a lot of user engagement, as it allows users to explore neat photos about space which will want users to engage more with the website.

Recommendation to Improve The Website

I would likely try to fix the missing text on the links and the buttons. Basically what the accessibility checker imformed me of, as this website is honestly really well put together and it's hard to think of a good recommendation.

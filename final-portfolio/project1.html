What Did I Do In This Lab

Lab 5 showed me how to properly build a layout that worked on both a desktop and mobile experience. I accomplished this by using flexbox, and reusing the information provided from the previous labs. Overall it was quite a fun lab to develop!

What I Learned From This Lab

How Did I Accomplish This?

Through the following code I was able to implement the Flexbox and enhance a mobile users experience!

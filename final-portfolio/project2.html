What Did I Do In This Lab

In Lab 7 I put together a tier list with a table and images! This lab assignment was very fun to me as I realized how easy tables were to create and how much can be done with the implementation of them.

What I Learned From This Lab

  • I learned how to properly implement tables into my HTML page
  • I learned how to create a contact form

How Did I Accomplish This?

<h1>Contact Form For SpongeBob Tier List</h1>
<form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="tier">Whats Your Favorite Tier:</label>
    <select id="tier" name="tier">
        <option value="s">S</option>
        <option value="a">A</option>
        <option value="b">B</option>
        <option value="c">C</option>
        <option value="d">D</option>
    </select>

    <label>Do you agree with the tier list at all?</label>
    <input type="radio" id="agree" name="opinion" value="agree">
    <label for="agree">Yes</label>
    <input type="radio" id="disagree" name="opinion" value="disagree">
    <label for="disagree">No</label>
